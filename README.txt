目前只支持自动个人信息回复，如有特殊需求联系作者qq:1240505995

需要安装 itchat   [安装命令：pip install itchat]

配置“config.ini”里的配置项
    “receiveGroup”设置为 1 表示接受群组信息
    “receivePerson”设置为 1 表示接受个人信息
    “replyGroup”设置为 1 表示回复所有群组的信息，
                   设置为['Group','Group',..] 回复特定群组
    “replyPerson”设置为 1 表示回复所有人的信息，
                   设置为['person1','person2',..] 回复特定人群
    “saveGroup”:设置为 1 表示保存群聊信息到本地
